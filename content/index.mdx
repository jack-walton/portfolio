# Introduction

I built this site to collect various writing samples and projects that are outside of my day job. I also wanted to challenge myself to build a documentation site from the ground up. To learn more about my real-world, shipped projects, see [my about page](https://jackwalton.net/about).

The [AXS User Guide](/AXS) is a speculative project. I am not affiliated with AXS. The guide covers account setup, navigation, event search, and ticket purchase.

The [Prompt Engineering](/AI) section documents a structured prompt I use with Claude Code to generate the site's change log.

[Music Information Retrieval in Bandcamp](/bandcamp) is a paper I wrote while I was a graduate student in Library and Information Science at the University of Missouri. In that paper, I conduct an analysis of the Bandcamp search function and compare it to peer-reviewed UX literature to develop a set of best practices for content managers.
