# Search for Events

Complete the following steps to search for events.

<Steps>

## Start Your Search

From the top of the **Search** screen, select **Search by Events, Performers or Venues**. In the search bar, enter an event, performer, or venue.

![The Search screen with the search bar at the top](/images/AXS/search_for_events/search_for_events_step_1.jpg)

## Select a Time

Select the **Time** box. Select one of the quick options, or press **Select Custom Date** and select a date from the calendar.

![The Time selection screen with quick date options and a Select Custom Date button](/images/AXS/search_for_events/search_for_events_step_2.jpg)

## Select a Venue

Select the **Location** box. Enter a city, select **Use Current Location**, or choose from a list of recent locations.

![The Location selection screen with options to enter a city or use the current location](/images/AXS/search_for_events/search_for_events_step_3.jpg)

## Refine Your Search

You can choose a performer from the **Performers** section to view a list of events by that performer.

![The Performers section of the search results](/images/AXS/search_for_events/search_for_events_step_4.0.jpg)

You can also choose a venue from the **Venues** section to see a list of events at that venue.

![The Venues section of the search results](/images/AXS/search_for_events/search_for_events_step_4.1.jpg)

You can also select an event from the **Recommended Events** section. Otherwise, select **Search** at the bottom of the App. Your search results are sorted into three tabs, **Events**, **Performers**, and **Venues**. Select a tab and scroll through the results.

![Search results organized into Events, Performers, and Venues tabs](/images/AXS/search_for_events/search_for_events_step_10.jpg)

## Select Event

When you have found an event, select **Buy Now** to purchase the tickets. See [Buy Tickets](/AXS/buy-tickets) to learn more.

![An event listing with a Buy Now button](/images/AXS/search_for_events/search_for_events_step_11.jpg)

</Steps>
